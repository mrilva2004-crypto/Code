## Описание программы

Weather Diary - это desktop-приложение с графическим интерфейсом для ведения дневника наблюдений за погодой. Программа позволяет добавлять, просматривать, редактировать и удалять записи о погоде, а также фильтровать их по различным критериям.

### Основные возможности

- 📝 **Добавление записей** с указанием даты, температуры, описания и наличия осадков
- 🔍 **Фильтрация записей**:
  - По конкретной дате
  - По температуре (выше +10°C / ниже или равно +10°C)
- ✏️ **Редактирование** существующих записей
- 🗑 **Удаление** записей
- 💾 **Автоматическое сохранение** всех записей в JSON-файл
- 📊 **Статистика** по отображаемым записям (количество, средняя/мин/макс температура, дни с осадками)
- 🖱️ **Удобный интерфейс** с прокруткой и таблицей
