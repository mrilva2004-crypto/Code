# GitHub User Finder

## Автор
**Сычук Александр** (Alexander Sychuk)

## Описание
GitHub User Finder - это desktop-приложение с графическим интерфейсом для поиска пользователей GitHub, просмотра результатов поиска и сохранения избранных пользователей в локальное хранилище.

## Возможности
- 🔍 Поиск пользователей GitHub по логину
- 📋 Отображение результатов поиска в таблице
- ⭐ Добавление пользователей в избранное
- 💾 Автоматическое сохранение избранного в JSON-файл
- ❌ Удаление пользователей из избранного
- 📱 Простой и интуитивно понятный интерфейс
